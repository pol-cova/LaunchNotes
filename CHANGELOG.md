# Changelog

All notable changes to LaunchNotes will be documented in this file.

The format follows a simple, human-readable structure. This project uses semantic versioning once public tags are created.

## 0.1.0 - 2026-05-31

Initial public release.

### Added

- SwiftUI release-note presentation for iOS 26 and macOS 26.
- Automatic version tracking with `UserDefaults`.
- Manual presentation with `Binding<Bool>`.
- Sheet and full-screen presentation modes.
- `LaunchNote` and `LaunchNotesRelease` models.
- Note and release result builders.
- Bundle version helper with `LaunchNotesVersion.current()`.
- Style presets and custom `LaunchNotesStyle`.
- Release-level accent color overrides.
- Optional footer action support.
- iOS demo app and package tests.
